# Calendar App

This is a demo app of a calendar made with
 - Vite
 - TS
 - Redux
 - Vitest
 - React-Router
 - Axios
 - Bootstrap

> [!NOTE]
> Before run dev, run the backend and setup env variables `VITE_API_URL`

## Dev

Run `npm run dev`

## Build

Run `npm run build`
